list.length()
    Return the number of elements in the list.

list.append(x)
    Add an item to the end of the list.

list.pop()
    Remove the last item in the list and return it.

list.search(x)
    Return the index of the first item whose value is x in the list. Return -1 if there is no such item.

list.getitem(i)
    Return the item of index i.

list.setitem(i, x)
    Change the item of index i in the list to value x.

list.insert(i, x)
    Insert an item before the element of index i. For example, a.insert(0, x) inserts at the front of the list, and a.insert(a.length(), x) is equivalent to a.append(x).

list.remove(i)
    Remove the item at the given position in the list, and return it.

list.extend(L)
    Extend the list by appending all the items in the given list L.

list.count(x)
    Return the number of times x appears in the list.

list.reverse()
    Reverse the elements of the list in place.

list.clear()
    Remove all items from the list.
